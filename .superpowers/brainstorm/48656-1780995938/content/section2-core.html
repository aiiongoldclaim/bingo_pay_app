Section 2 — Core Module Architecture

Every shared service lives here. Features depend on core, never on each other.

Network

core/api/

api_client.dart
api_endpoints.dart
interceptors/

 ├─ logging_interceptor.dart
 └─ error_interceptor.dart
JWT auto-attach • token refresh • request logging

core/config/

Flavor-aware app configuration

flavor_config.dart
app_config.dart

API URLs • feature flags • timeouts • pagination sizes

core/di/

injection.dart

Single service locator • env-aware registration

app_router.dart
app_routes.dart

Shared Widgets

core/widgets/

Design system components

Dependency Rule: Features → Core. Core → nothing in features. Features never import each other.
