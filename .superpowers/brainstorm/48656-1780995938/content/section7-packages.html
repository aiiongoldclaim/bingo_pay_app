Testing strategy

Unit tests — test/unit/

UseCase, Repository, Failure mapping. Mock datasources with mocktail. No Flutter dependency. Run in <5s.

BLoC tests — test/bloc/

Use bloc_test + whenListen. Verify state sequences: Loading → Success, Loading → Error. 100% BLoC coverage target.

Widget tests — test/widget/

Pump screens with mocked BLoC states. Verify UI renders correctly for Loading / Success / Error states. Test role-based widget visibility.

Integration tests — integration_test/

Full login → home flow per flavor. KYC upload flow. Run on real device / emulator via flutter test integration_test/.
