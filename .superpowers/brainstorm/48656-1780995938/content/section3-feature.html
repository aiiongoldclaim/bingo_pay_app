Data flow (login example)

1. Screen — user taps Login → dispatches LoginRequested event to AuthBloc
↓
2. BLoC — calls LoginUseCase(email, password)
↓
3. UseCase — calls AuthRepository.login(), returns Either<Failure, UserEntity>
↓
4. Repository Impl — calls AuthRemoteDataSource.login() → maps UserModel → UserEntity
↓
5. Remote DS — Dio POST /auth/login → returns AuthResponseModel, saves JWT to SecureStorage
↓
6. BLoC emits AuthSuccess(user) → GoRouter redirects based on user.role
